# Tasks Alarm

Приложение проговаривает голосом задачи на сегодня, созданные в приложении TickTick, в момент показа ежедневого уведомления (например, в 9 утра).
Есть возможность остановить проговаривание нажатием на само уведомление или на кнопку Стоп.

## Основные технологии
- Google Cloud Text to Speech
- NotificationListenerService
- TickTick Content Provider
- Clean Architecture
- Koin
- Firebase Analytics
- Crashlytics
