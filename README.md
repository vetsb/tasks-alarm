# Tasks Alarm

Приложение проговаривает голосом задачи на сегодня, созданные в приложении TickTick, в момент показа ежедневого уведомления.

## Основные технологии
- Google Cloud Text to Speech
- NotificationListenerService
- TickTick Content Provider
- Clean Architecture
- Koin
- MVVM
- Firebase Analytics
- Crashlytics
